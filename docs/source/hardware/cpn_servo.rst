舵机
===========

舵机是一种特殊的电机，以精确控制特定角度位置而闻名。

.. image:: img/servo.png
    :align: center

* **棕线** ：GND
* **橙线** ：信号引脚，连接到主板的 PWM 引脚。
* **红线** ：VCC

与持续旋转的普通电机不同，舵机可以移动到精确位置并精确保持。这是通过齿轮组、电位器和控制电路的组合实现的。舵机通常用于各种需要精确控制物体或机构位置的应用中。

**特性**

* **电机类型** ： Core 电机
* **工作电压** ：4.8~6V 直流
* **待机电流** ：≤4 mA
* **消耗电流（4.8V 空载）** ：≦50mA
* **消耗电流（6V 空载）** ：≦60mA
* **堵转电流（4.8V 锁定）** ：≦550mA
* **堵转电流（6V 锁定）** ：≦650mA
* **额定扭矩** ：4.8V，≥0.6 kgf·cm；6V，≥0.7 kgf·cm
* **最大扭矩** ：4.8V，≥1.4 kgf.cm；6V，≥1.6 kgf.cm
* **空载速度** ：4.8V，≦0.14秒/60°；6V，≦0.12秒/60°
* **注意** ：扭力保护：≥0.9 kgf.cm；连续 5 秒后断电保护
* **工作温度范围** ：-10℃~+50℃
* **存储温度范围** ：-20℃~+60℃
* **工作湿度范围** ：≤ 90%RH
* **存储湿度范围** ：≤ 90%RH
* **重量** ：10± 0.5g
* **材质** ：ABS
* **工作角度** ：180°±10°（500~2500us）
* **机械极限角度** ：360°
* **左右行程角度** ：≤ 6°
* **居中偏差** ：≤ 1°
* **回差** ：≤ 4 us
* **放大器类型** ：数字
* **线材材质** ：Ф1.08,19 PVC
* **线长** ：245±5mm（不含端子）
* **连接器类型** ：JR2.54mm/3Pin

**工作原理**

舵机内部，关键组件共同构成了其独特的功能。舵机的核心是一个传统电机，该电机与一个大齿轮精密连接，大齿轮又与电机轴上的一个小齿轮啮合。这种齿轮传动装置有效地将电机的快速圆周运动转换为较慢但强大的运动。

.. image:: img/servo_internal.png

但真正的神奇之处在于舵机内部，这要归功于一个被称为 "电位器" 的微小电子奇迹和精密的 "控制电路"。当舵机运动时，电位器旋转，改变其电阻。控制电路以卓越的精度检测并解释这种电阻变化，从而确定舵机的精确位置。这证明了其设计的巧妙性。

在舵机控制领域，一种称为 "脉宽调制" 或 PWM 的独特信号方法发挥作用。通过巧妙地调整这些脉冲的宽度，操作者可以指令舵机精确移动并保持其位置。这就是舵机电机技术的精髓，精度与控制在这里交汇，实现了各种应用。
