火星の荒涼とした地形を走行し、未知の世界を探検するローバーを操縦することを想像してみてください。NASAのエンジニアにとっては夢のような話でしょうか？

もう違います。

ようこそ、「GalaxyRVR」へ。SunFounderが提供するこのマーズローバーキットは、惑星探査をあなたのリビングルームに届けます。

.. image:: img/galaxyRVR.jpg
    :width: 500
    :align: center

NASAのロッカー・ボギーシステムに基づいて構築されたGalaxyRVRは、岩場の山、砂浜、草原をスムーズに移動でき、まるで火星が身近に感じられるでしょう。

高解像度カメラを搭載したGalaxyRVRは、一人称視点で没入感のある操縦体験を提供します。その高度な障害物回避機能と超音波モジュールにより、障害物を自動で回避することが可能です。

あなたの興味やスキルレベルに応じてコースを選べます：

* :ref:`quick_start` - モバイルアプリを使用してGalaxyRVRをすぐに操作可能。手軽に楽しみたい初心者に最適です。

* :ref:`programming_arduino` - コードを深く掘り下げて各機能を理解し、Arduinoプログラミングを通じてGalaxyRVRをカスタマイズ。知識を深めたい、実験を楽しみたい方に最適です。
