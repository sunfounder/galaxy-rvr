玩转 Scratch
============================

除了在 Arduino IDE 上编程，我们还可以使用图形化编程。

我们推荐使用 Scratch 进行编程。
然而，官方的 Scratch 目前仅支持 Raspberry Pi。
为了解决这个问题，我们开发了 Mammoth Coding，这是一款基于 Scratch 3 的图形化编程软件，专门为 Arduino 板（Uno、Mega2560 和 Nano）设计。

Mammoth Coding 保留了 Scratch 3 的基本功能，并新增了控制 Arduino Uno、Mega、Nano 等自定义板的能力。它允许你使用外部传感器和机器人来控制舞台上的精灵，提供强大的硬件交互功能。

此外，即使你没有太多编程经验，这款应用也非常容易上手。你可以轻松地学习和应用这些前沿技术。

只需拖放 Scratch 编程模块，即可创建激动人心的游戏、动画、互动项目，甚至根据你的构想控制机器人！
