第7课：创建红外避障动画
=====================================================

在之前的任务中，我们使用红外避障模块帮助 GalaxyRVR 自动避开障碍物——就像真正的火星车在红色星球上导航一样！

现在，让我们更进一步，将物理红外传感器与虚拟的火星景观结合起来。我们将创建一个激动人心的动画游戏，用手触发真实红外传感器来控制火星车精灵！

学习目标
-------------------------

* 发现红外避障模块如何让你的火星车生动起来
* 学习使用红外传感器数据控制 Scratch 动画中的角色
* 构建你自己的火星探索游戏，使用真实传感器躲避岩石

创建动画场景
----------------------------

#. 首先，请执行 :ref:`app_connect` 的步骤。

#. 设置火星主题背景

   * 首先，我们需要一个火星主题的舞台背景。点击选择一个背景。

     .. image:: img/5_animate_choose.png

   * 选择 **火星** 背景。

     .. image:: img/5_animate_mars.png

#. 添加火星车角色

   从库中选择 **GalaxyRVR** 精灵，并适当调整其大小以适应场景。

   .. image:: img/5_animate_rvr.png

#. 添加障碍物

   从库中选择一个 **岩石** 精灵并调整其大小。

   .. image:: img/5_animate_rock.png


编程 GalaxyRVR 精灵
------------------------------------------

编程让 GalaxyRVR 精灵从屏幕左侧移动到右侧。你的任务：引导它安全地穿过岩石！
用手触发物理火星车的红外传感器——右侧传感器使精灵向下移动，左侧传感器使其向上移动。

#. 通过将精灵移动到舞台左边缘来设置起始位置。运动积木块将自动更新为正确的坐标。

   .. image:: img/5_animate_glide.png

#. 添加一个重复执行积木块，创建持续运行的主程序循环。

   .. image:: img/5_animate_forever.png

#. 包含一个条件积木块，检查火星车是否碰到岩石障碍物。

   .. image:: img/5_animate_touching.png

#. 如果没有检测到岩石，则继续向右移动。

   .. image:: img/5_animate_moving.png

#. 如果火星车撞到岩石，停止所有移动并显示警告信息。

   .. image:: img/5_animate_say.png

#. 添加传感器控制：为两个红外传感器创建事件，当用手触发时，左侧传感器使精灵向上移动，右侧传感器使其向下移动。

   .. image:: img/5_animate_y.png

编程岩石精灵
------------------------------------

让我们创建多个岩石障碍物，让游戏更有挑战性！我们将使用克隆功能在舞台的随机位置生成岩石。

#. 使用"创建自己的克隆"积木块创建岩石克隆体。

   .. image:: img/5_animate_clone.png

#. 添加"移动到随机位置"积木块，让每个克隆体出现在随机位置。

   .. image:: img/5_animate_clone_move.png
      :width: 200

#. 通过重复执行克隆创建十次，生成十块岩石。

   .. image:: img/5_animate_clone_10.png
      :width: 200

#. 将所有操作设置为在点击绿旗时启动。

   .. image:: img/5_animate_clone_flag.png
      :width: 200

任务发射！
----------------------------

恭喜！你的火星避障游戏已准备好发射。

将 GalaxyRVR 连接到应用，点击绿旗开始你的任务。观察岩石随机出现在火星景观中。

你的挑战：用手触发红外传感器，小心地引导 GalaxyRVR 精灵穿过屏幕。上下移动以避开岩石，安全抵达右侧！

你能在没有任何碰撞的情况下完成任务吗？你能多快穿越障碍路线？继续练习，成为火星车驾驶大师吧！
