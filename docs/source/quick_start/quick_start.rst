快速开始
===================

本章帮助您快速开始使用您的 GalaxyRVR。
您可以通过 RoboPilot App 控制它，或在 Arduino IDE 或 Mammoth Coding 中运行现成的示例代码——无需编程。

GalaxyRVR 支持三种控制方式：

- **RoboPilot App** ：从您的移动设备驾驶和操作 GalaxyRVR。
- **Arduino IDE** ：打开并上传示例代码来控制火星车。
- **Mammoth Coding** ：使用拖放式积木块加载并运行示例项目。

您可以选择以上任意一种方式开始控制您的 GalaxyRVR。
