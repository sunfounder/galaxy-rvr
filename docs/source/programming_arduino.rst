.. _programming_arduino:

Arduino IDE を使ったプログラミング
====================================

火星ローバーの内部構造を探り、その機能をカスタマイズするための13レッスン構成の学習旅をお楽しみください。火星ローバーの歴史から始め、主要コンポーネントの組み立てと統合を実践的に学びます。

動画、実践レッスン、詳細なガイドを通じて、GalaxyRVRを動かす機械、電子、ソフトウェアの包括的な理解を深めます。最終的には、完全に組み立てられたローバーとトラブルシューティング、改造、そして革新を行うスキルを手に入れることができます。

赤い惑星を身近に感じる準備をしましょう—ローバーの冒険を今すぐ始めてください！

**コードをダウンロード**

以下のリンクから関連するコードをダウンロードしてください。

    * :download:`GalaxyRVR コードファイル <https://github.com/sunfounder/galaxy-rvr/archive/refs/heads/main.zip>`

または、 `SunFounder GalaxyRVR Kit for Arduino - GitHub <https://github.com/sunfounder/galaxy-rvr/tree/main>`_ でコードを確認してください。
